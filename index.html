Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

I would advise the client to use multiple ways to secure the website from attackers. this should be using https instead of http. Https uses encryption to ensure a secure communication between server and client.
Html sanitizing should be implemented on all the user data to ensure that it is more difficult to input malicious code in user forms. A web application firewall can also be used to sanitize all the data between user and the web page.
The website should encourage the users to use strong passwords and even better if they use two-factor authentication.
